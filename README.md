Uhtred
======================

The frontend (React.js version) for [Marketr](https://sonataxa.tech).

Tech
--------
  * ES6
  * Webpack
  * React + Redux
  * Jest + Enzyme

Setup & Running locally
--------
	$ npm install
	$ npm run build (webpack)
	$ npm run server:dev (webpack-dev-server --inline --hot)

Testing
--------
	$ npm run tdd (jest --watch)
