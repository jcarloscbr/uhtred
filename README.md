Uhtred
======================

The frontend (React.js version) for [Marketr](https://sonataxa.tech).

Setup & Running locally
--------
	$ npm install
	$ webpack
  $ webpack-dev-server (runs at localhost:8080)

Tech
--------
  * ES6
  * Webpack
  * React + Redux
